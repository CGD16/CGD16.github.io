Uno

In order to improve my Python programming skills I wrote this project-game "UNO". With OOP (Object - Oriented Programming) I could extend the game to n-player game.

Uno is played in several rounds. At the beginning of each round each player has n cards (you can specify n). During the game, the cards can added to the center-card if the player has the same color or the same number. The player who has no cards left in the hand wins the round
